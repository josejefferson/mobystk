Configurar emuladores
  • Antigo

O MobyStk pode configurar as teclas automaticamente para alguns emuladores. Siga as instruções de cada emulador para configurá-lo. Prepare o emulador e clique no botão vermelho para iniciar.

O IP está definido como ... (status: ).
Para mudar o IP, volte à tela inicial, altere e abra o Joystick, depois volte aqui.

PPSSPP

Entre em Configurações > Controles > Mapear controles

Snes9x

Entre em Input > Input Configuration... > (Selecione a opção Up)

PCSX2

Entre em Configuração > Comandos (PAD) > Configurações do plug-in... > Pad (1 ou 2) > Quick setup
